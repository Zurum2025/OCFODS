# Online Campus Food Ordering and Delivery System

An online platform that allows students to order food from campus vendors,
with features for vendor management and administrative control.

This project is being developed as a full-stack web application
using Flask, following object-oriented design and UML-based system analysis.

---

## Features (Current)
- Landing page
- login page
- registration page
- Session-based authentication
- Base template with shared layout
- Basic UI styling

---

## Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, Jinja2
- **Version Control:** Git & GitHub
